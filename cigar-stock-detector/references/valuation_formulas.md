# 估值公式详解

## 概览

本文档详细说明烟蒂股探测的核心估值公式、参数选择依据和边界条件。所有公式均经过实战验证，具备专家级应用价值。

## 核心公式

### T0: 净现金法

**公式**：
$$T0 = \frac{(\text{现金池} + \text{FVTOCI资产} + \text{债权投资CDs}) - (\text{总负债} - \text{合同负债})}{\text{总股本}}$$

**含义**：每股净现金价值，最保守的资产垫。

**现金池扩展项**：
| 扩展项 | 系数 | 穿透要求 |
|--------|------|---------|
| FVTOCI中高流动性债券 | 1.0 | 需穿透确认以国债/政策性金融债为主 |
| 债权投资中的大额存单/CDs | 1.0 | 需穿透确认存单性质 |

**合同负债剔除规则**：
- 合同负债本质是预收款，无需现金偿还，只会转化为未来收入
- T0计算中允许从总负债中剔除，还原真实债务负担
- 剔除后需在报告中标注，避免误导

**适用场景**：
- 现金占比极高(> 总资产50%)
- 负债结构简单
- 清算可能性大

**投资标准**：股价 ≤ T0 × 0.7(安全边际30%)

---

### T1: 有息负债扣除法

**公式**：
$$T1 = \frac{\text{现金池} + \text{FVTOCI资产} + \text{债权投资CDs} - \text{有息负债}}{\text{总股本}}$$

**含义**：扣除真实债务负担后的每股净现金。

**有息负债包括（必须强制计入）**：
| 科目 | 准则 | 备注 |
|------|------|------|
| 短期借款 | 全准则 | 银行借款等计息负债 |
| 长期借款 | 全准则 | 中长期银行借款 |
| 应付债券 | 全准则 | 企业债、公司债 |
| 一年内到期的非流动负债 | 全准则 | 借款部分 |
| 租赁负债 | 全准则 | 经营租赁/融资租赁 |
| **应付票据(商业承兑)** | **CN GAAP** | **企业自身信用开票，本质短期融资** |
| **永续债** | **全准则** | **虽在权益列示，但对普通股属债务** |

**应付票据细化处理**：
- 银行承兑汇票：有银行信用担保，不视同有息负债
- 商业承兑汇票：企业自身信用开票，本质6-12个月短期融资，**必须计入有息负债**
- 未区分类型时：保守处理，全部计入有息负债

**重要说明**：
- T1不剔除合同负债（有息负债本身不包含合同负债，无需剔除）
- FVTOCI和CDs同样加入T1现金池

**适用场景**：
- 无息负债占比较高(如应付账款)
- 经营性负债良性
- 中等保守评估

**投资标准**：股价 ≤ T1 × 0.8(安全边际20%)

---

### T2: 清算价值法（动态系数版）

**公式**：
$$T2 = \frac{\text{现金池} \times 1.0 + \text{FVTOCI} \times 0.9 + \text{CDs} \times 1.0 + \text{AR} \times 0.85 + \text{其他AR} \times 0.2 + \text{存货} \times \text{动态系数} - \text{总负债}}{\text{总股本}}$$

**含义**：模拟清算场景下的每股回收价值。

**折扣系数依据（专家级颗粒度）**：

| 资产类型 | 折扣系数 | 理由 |
|---------|---------|------|
| 现金 | 1.0 | 完全变现 |
| **FVTOCI高流动性债券** | **0.9** | **清算时可能折价** |
| **债权投资CDs** | **1.0** | **本质是现金** |
| 应收账款 | 0.85 | 坏账损失、催收成本 |
| **其他应收款** | **0.2** | **常见雷区，大股东占用、虚假资产** |
| **存货（动态系数）** | **见下表** | **按行业变现能力差异化** |

**动态存货系数表**：

| 行业类型 | 存货系数 | 典型行业 | 理由 |
|---------|---------|---------|------|
| 硬通货 | 0.8 - 0.9 | 白酒、贵金属、部分原材料 | 变现能力强，甚至增值 |
| 一般制造/房地产 | 0.6 - 0.7 | 传统制造、地产 | 存在跌价风险，但可控 |
| 电子产品/时尚 | 0.3 - 0.5 | 消费电子、服装、食品 | 跌价极快，时效性强 |
| 高风险库存 | 0.1 - 0.2 | 定制化设备、专用零部件 | 清算基本归零 |

**T2合同负债处理**：
- T2使用完整总负债（保守计入），不剔除合同负债
- 理由：清算场景下，预收款对应的未来交付义务构成真实负债

**其他应收款处理说明**：
- 其他应收款是烟蒂股最常见的藏雷点
- 系数设为0.2而非0，保留最低回收可能
- 若发现关联方占用或异常增长，应系数设为0并触发VETO

**适用场景**：
- 资产结构均衡
- 存货和应收账款质量可接受
- 需全面评估资产价值

**投资标准**：股价 ≤ T2 × 0.9(安全边际10%)

## 资产烧损率（优化版）

**公式**：
$$\text{资产烧损率} = \frac{\max(0, -FCF)}{\text{资产垫} \times \text{总股本}}$$

**定义**：
- FCF：自由现金流 = 经营活动现金流净额 - 资本支出
- 资产垫：T0/T1/T2 对应的总资产垫(取最大正值)
- max(0, -FCF)：仅当FCF为负时计算，FCF为正时烧损率为0

**含义**：企业每年烧钱速度占资产垫的比例。修正版消除了FCF为正时的计算缺陷。

**阈值（专家级标准）**：
- < 5%：安全线，资产垫极其稳固
- 5% ~ 20%：预警线，资产垫正在被侵蚀，需配合兑现路径评估
- ≥ 20%：否决线，价值毁灭型，除非有极强催化剂，否则直接否决

**阈值调整理由**：
- 原阈值100%太宽松，一年烧完资产垫已是濒死状态
- ≥20%意味着5年内资产垫将清零，这是合理的安全边界

**示例**：
```
案例1：烧钱企业
资产垫 = 10亿元，总股本 = 10亿股
年度FCF = -8亿元
资产烧损率 = max(0, 8) / (10 × 10) = 8% → 预警线

案例2：现金牛企业
资产垫 = 10亿元
年度FCF = +5亿元
资产烧损率 = max(0, -5) / 100 = 0 → 安全，触发加分项
```

## 特殊科目处理规程（Boundary Conditions）

### 核心原则：烟蒂股清算逻辑

| 科目 | 处理动作 | 理由 | 影响 |
|------|---------|------|------|
| **受限现金** | 从现金池中100%扣除 | 无法动用的现金无保护意义 | 降低T0/T1/T2 |
| **合同负债** | T0剔除(从总负债减去)，T1不影响，T2保守计入 | 本质是预收款，无需现金偿还 | T0提升 |
| **商誉** | T0/T1/T2统一记为0 | 清算场景下无形资产价值归零 | 仅影响风险扫描 |
| **不可辨认无形资产** | T0/T1/T2统一记为0 | 同上 | 同上 |
| **可辨认无形资产(土地使用权)** | T2可给0.3系数 | 清算时可能有一定变现价值 | 谨慎计入T2 |
| **其他非流动资产** | 需穿透，通常系数≤0.1 | 复杂资产，清算价值低 | 需人工判断 |
| **FVTOCI高流动性债券** | T0/T1系数1.0，T2系数0.9 | 优质T0资产，清算时微折 | 提升T0/T1 |
| **债权投资CDs** | T0/T1/T2系数1.0 | 本质是现金 | 全面提升 |

### 受限现金穿透要点

受限现金类型（必须识别）：
- 银行承兑汇票保证金（业务性，可解付）
- 信用证保证金（业务性，可解付）
- 按揭保证金（业务性，长期受限）
- 司法冻结资金（风险性，直接扣除）
- 专项用途资金（风险性，直接扣除）

**实战经验**：受限现金占比 > 15%时，必须深入穿透；> 50%时直接否决。

## 资产垫分级

### 分级标准

| 级别 | 判定条件 | 安全边际 | 风险等级 | 典型特征 |
|------|---------|---------|---------|---------|
| T0 | 股价 ≤ T0 × 0.7 | 30% | 极低 | 现金牛，保守选股 |
| T1 | 股价 ≤ T1 × 0.8 | 20% | 低 | 净现金，适度扩张 |
| T2 | 股价 ≤ T2 × 0.9 | 10% | 中 | 均衡资产，需质量判断 |
| 未通过 | 不满足上述条件 | - | 高 | 不符合烟蒂股标准 |

**优先级**：T0 > T1 > T2(优先选择更保守的估值)

## 兑现路径判定（量化硬指标）

### A型：现金分红型

**定性特征**：T0/T1级资产垫 + 股息率 > 5%

**量化准入门槛**：
- ✅ **现金池 / 年度总派息 > 1.5**
- ✅ 连续3年分红且递增
- ✅ 经营现金流/净利润 > 100%

**含义**：确保公司账面现金足够支撑未来分红，而非透支或借款分红。

**兑现路径**：持续分红回收投资
**持有期**：3-5年
**风险点**：分红政策突变

---

### B型：分拆清算型

**定性特征**：子公司上市 + SOTP折价 > 30%

**量化准入门槛**：
- ✅ **持股覆盖率 > 50%**
  - 定义：(所持子公司股权市值 × 持股比例) / 母公司市值 > 50%
- ✅ 子公司流动性良好(日均成交额 > 母公司市值1%)
- ✅ 分拆政策障碍低

**含义**：持股覆盖率<50%时，"增强套利"杠杆力不足，不足以驱动价值重估。

**兑现路径**：子公司分拆、资产出售、控股权转让
**触发条件**：控股股东变更、战略调整、监管放宽
**风险点**：子公司估值同步下跌

---

### C型：均值回归型

**定性特征**：T2级资产垫 + 行业周期底部

**量化准入门槛**：
- ✅ PB < 1.5（行业均值）
- ✅ 行业产能利用率 < 70%
- ✅ 龙头企业份额 > 20%

**兑现路径**：行业回暖、估值修复
**风险**：周期判断错误、长期衰退

---

## 加分项说明

### 国企背景(+1分)
- 控股股东为国资委或地方政府
- 融资渠道畅通
- 政策支持力度大
- 退市风险低

### 子公司持股(+1分)
- 持有上市公司股权
- 市值可实时查询
- SOTP折价明显(>30%)
- 分拆可能性大

### 分红记录良好(+1分)
- 连续3年分红且递增
- 股息率 > 5%
- 分红比例稳定(30%-50%)
- 现金流/净利润 > 100%
- 现金池/派息 > 1.5

## 一票否决项

在Fact Check扫描中，以下任一条件触发时，**直接否决**，不进入后续分析：

| 否决项 | 判定标准 | 风险描述 |
|--------|---------|---------|
| **V1: 持续经营假设存疑** | 审计报告含"持续经营重大不确定性" | 濒临破产 |
| **V2: 审计意见为否定/无法表示** | 否定意见、无法表示意见 | 财务数据不可信 |
| **V3: 核心资产被查封** | 核心经营资产被司法冻结 | 资产无法变现 |
| **V4: 大股东资金占用** | 其他应收款中控股股东占用 > 净资产10% | 利益输送 |
| **V5: 重大诉讼未披露** | 涉案金额 > 净资产20%且未披露 | 潜在巨额损失 |
| **V6: 财务造假历史** | 证监会处罚历史 | 诚信破产 |
| **V7: 受限现金占比过高** | 受限现金/货币资金 > 50% | 虚假现金 |
| **V8: 连续亏损无改善** | 连续3年亏损且无扭亏计划 | 持续价值毁灭 |
| **V9: 资产烧损率≥20%** | 年度烧损率 ≥ 20% | 5年内资产垫清零 |
| **V10: 大股东质押率>70%** | 大股东质押股份比例 > 70% | 强平风险 |
| **V11: 关联交易占比>30%** | 关联交易/营业收入 > 30% | 利益输送嫌疑 |
| **V12: 审计意见为保留意见** | 保留意见(Qualified Opinion) | **新增：账本基础被动摇，直接否决** |

## 边界条件

### 数据单位一致性
所有金额单位需统一(亿元或万元)，股本单位统一(亿股或万股)。

### 负值处理
- T0/T1/T2为负值时，资产垫为0
- 总负债 > 现金池时，T0为负
- 有息负债 > 现金池时，T1为负

### 缺失数据处理（保守降级原则）

- 缺少股价：仅计算NAV，不评级
- 缺少FCF：不计算资产烧损率
- 缺少AR/存货/行业：使用T0/T1公式
- 缺少行业类型：存货系数默认0.6（保守处理）

**防幻觉降级规则（必须严格执行）**：

| 缺失场景 | 降级处理 | 输出标记 |
|---------|---------|---------|
| 交易性金融资产明细无法穿透 | 默认按0计入现金池 | [NULL-需人工核查] |
| FVTOCI资产构成不明 | 默认按0计入现金池 | [NULL-需人工核查] |
| 受限现金金额不明但附注提示质押 | 货币资金打8折计入 | [受限现金-8折估算] |
| 应付票据未区分银行/商业承兑 | 全部计入有息负债(保守) | [票据类型未知-保守处理] |
| 合同负债与预收款项混淆 | 视同合同负债处理 | [科目待确认] |
| 其他应收款前五大欠款方不明 | 系数直接降为0 | [NULL-需人工核查] |
| 存货行业类型不明 | 系数默认0.6 | [行业待确认] |

**核心原则**：
1. 宁可低估资产，不可高估
2. 宁可高估负债，不可遗漏
3. 任何估算值必须标注[估算]前缀
4. 任何缺失数据必须输出[NULL-需人工核查]，严禁LLM自行编造

## 使用示例

### 示例1：制造业烟蒂股（含合同负债与FVTOCI）

**输入数据**：
```
现金池：100亿元
FVTOCI高流动性债券：8亿元
债权投资CDs：5亿元
总负债：30亿元（含合同负债5亿元）
有息负债：20亿元（含商业承兑汇票3亿元）
总股本：10亿股
应收账款：15亿元
其他应收款：5亿元（需重点关注）
存货：20亿元
存货类型：一般制造（系数0.65）
当前股价：5.0港元
年度FCF：-6亿元
```

**计算结果**：
```
T0 = (100 + 8 + 5 - (30 - 5)) / 10 = 88 / 10 = 8.8港元
T1 = (100 + 8 + 5 - 20) / 10 = 93 / 10 = 9.3港元
T2 = (100 + 8×0.9 + 5 + 15×0.85 + 5×0.2 + 20×0.65 - 30) / 10
   = (100 + 7.2 + 5 + 12.75 + 1.0 + 13.0 - 30) / 10
   = 108.95 / 10 = 10.895港元

资产烧损率 = max(0, 6) / (10.895 × 10) = 5.51% → 预警线

股价 5.0港元：
- T0级：5.0 ≤ 8.8 × 0.7 = 6.16？否
- T1级：5.0 ≤ 9.3 × 0.8 = 7.44？是
- 资产垫分级：T1

溢价率：((9.3 - 5.0) / 5.0) × 100% = 86%
```

**结论**：T1级烟蒂股，安全边际86%，合同负债剔除后T0显著提升。资产烧损率5.5%处于预警线，需深入分析现金消耗原因。建议穿透其他应收款5亿元的性质，确认FVTOCI 8亿元的资产构成。

---

### 示例2：现金牛企业

**输入数据**：
```
现金池：50亿元
FVTOCI：0（无高流动性债券）
总负债：10亿元（含合同负债2亿元）
有息负债：0亿元
总股本：5亿股
当前股价：6.0元
年度FCF：+8亿元
年度派息：3亿元
```

**计算结果**：
```
T0 = (50 + 0 + 0 - (10 - 2)) / 5 = 42 / 5 = 8.4元
T1 = (50 + 0 + 0 - 0) / 5 = 10.0元

资产烧损率 = max(0, -8) / (10 × 5) = 0 → 安全

现金池/派息 = 50 / 3 = 16.7 > 1.5 ✅

股价 6.0元：
- T0级：6.0 ≤ 8.4 × 0.7 = 5.88？否
- T1级：6.0 ≤ 10.0 × 0.8 = 8.0？是
- 资产垫分级：T1

溢价率：((10.0 - 6.0) / 6.0) × 100% = 66.7%
```

**结论**：T1级现金牛烟蒂股，安全边际66.7%，资产烧损率为0且现金池/派息比率极高，符合A型兑现路径。合同负债2亿元剔除后T0从8.0提升至8.4。综合评级A。

### 示例3：防幻觉降级案例

**输入数据**：
```
现金池：货币资金30亿元（附注提示"部分资金用于质押"，但未披露具体金额）
FVTOCI：10亿元（附注仅披露总额，未穿透明细）
总负债：15亿元
有息负债：8亿元
总股本：5亿股
应收账款：12亿元
其他应收款：3亿元（前五大欠款方附注缺失）
存货：10亿元
存货类型：不明
当前股价：3.5元
年度FCF：-2亿元
```

**降级处理**：
