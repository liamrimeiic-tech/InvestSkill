# Fact Check 风险扫描清单

## 概览

本文档列出烟蒂股探测中必须扫描的 23 项风险指标，分为 WARNING-Risk(警告项)和 VETO(否决项)两类。

## WARNING-Risk (警告项，扣分)

### W1. 商誉与无形资产占比过高
- **指标**：(商誉 + 不可辨认无形资产) / 净资产
- **阈值**：> 20%
- **扣分**：-1分
- **处理**：需检查减值风险，评估收购对价合理性；不可辨认无形资产包括品牌、客户关系、专利等清算归零项

### W2. 应收账款账龄恶化
- **指标**：1年以上应收账款 / 应收账款总额
- **阈值**：> 30%
- **扣分**：-1分
- **处理**：检查坏账准备充足性，分析客户信用风险

### W3. 存货跌价准备不足
- **指标**：存货跌价准备 / 存货余额
- **阈值**：低于行业平均或历史水平
- **扣分**：-1分
- **处理**：对比同行业公司，检查存货构成

### W4. 预付款项异常增长
- **指标**：预付款项同比增长率
- **阈值**：> 50%且无合理解释
- **扣分**：-1分
- **处理**：穿透预付款性质，检查关联方往来

### W5. 关联方往来频繁
- **指标**：关联方应收应付款项占比
- **阈值**：> 总资产 10%
- **扣分**：-2分
- **处理**：检查关联交易公允性，识别资金占用

### W6. 审计意见带强调事项段
- **指标**：审计意见类型
- **阈值**：带强调事项段的无保留意见
- **扣分**：-1分
- **处理**：阅读强调事项，评估影响程度
- **说明**：保留意见已升级为V12否决项，此处仅处理强调事项段

### W7. 受限现金占比偏高
- **指标**：受限现金 / 货币资金
- **阈值**：15% ~ 50%
- **扣分**：-1分
- **处理**：穿透受限原因，评估可动用现金

### W8. 其他应收款占比高
- **指标**：其他应收款 / 总资产
- **阈值**：> 10%
- **扣分**：-2分（升级扣分，此科目是烟蒂股核心雷区）
- **处理**：检查款项性质，识别潜在坏账；必须穿透前五大欠款方是否关联方
- **升级说明**：原阈值5%过于宽松，10%更能识别真实风险；扣分从-1升级为-2

### W9. 长期待摊费用占比高
- **指标**：长期待摊费用 / 总资产
- **阈值**：> 5%
- **扣分**：-1分
- **处理**：评估摊销政策，识别资产虚增

### W10. 递延所得税资产占比高
- **指标**：递延所得税资产 / 总资产
- **阈值**：> 3%
- **扣分**：-1分
- **处理**：检查可抵扣亏损确认合理性

### W11. 资产减值损失持续
- **指标**：连续2年大额资产减值
- **阈值**：> 净利润 50%
- **扣分**：-2分
- **处理**：识别减值资产类型，评估持续性

### W12. 现金流与利润背离
- **指标**：经营现金流净额 / 净利润
- **阈值**：< 0.5 或持续为负
- **扣分**：-2分
- **处理**：分析背离原因，识别利润质量

## VETO (否决项，一票否决)

### V1. 持续经营假设存疑
- **指标**：审计报告中持续经营重大不确定性段落
- **判定**：存在即否决
- **处理**：直接排除，不进入分析流程

### V2. 审计意见为否定/无法表示
- **指标**：审计意见类型
- **判定**：否定意见或无法表示意见
- **处理**：直接排除

### V3. 核心资产被查封冻结
- **指标**：资产权利受限情况
- **判定**：核心经营资产被查封、冻结、抵押
- **处理**：评估对持续经营影响，严重者否决

### V4. 控股股东资金占用严重
- **指标**：其他应收款中控股股东占用
- **判定**：占用金额 > 净资产 10%
- **处理**：检查清欠计划，评估回收风险

### V5. 重大诉讼仲裁未披露
- **指标**：或有事项披露
- **判定**：重大诉讼(涉案金额 > 净资产 20%)未披露或披露不充分
- **处理**：评估潜在损失

### V6. 财务造假历史
- **指标**：历史处罚记录
- **判定**：证监会处罚、虚假陈述诉讼
- **处理**：直接排除

### V7. 受限现金占比过高
- **指标**：受限现金 / 货币资金
- **判定**：> 50%
- **处理**：虚假现金风险极高，否决

### V8. 连续亏损且无改善迹象
- **指标**：连续亏损年数 + 扭亏措施
- **判定**：连续3年亏损且无明确扭亏计划
- **处理**：评估持续经营能力

### V9. 资产烧损率超标（专家级标准）
- **指标**：资产烧损率 = max(0, -FCF) / 资产垫
- **判定**：≥ 20%（优化阈值，原100%过于宽松）
- **处理**：5年内资产垫将清零，价值毁灭型，除非有极强催化剂，否则否决

### V10. 大股东质押率过高
- **指标**：大股东质押股份比例
- **判定**：> 70%
- **处理**：强平风险极高，直接否决

### V11. 关联交易占比过高
- **指标**：关联交易 / 营业收入
- **判定**：> 30%
- **处理**：利益输送嫌疑，财务可信度存疑，直接否决

### V12. 审计意见为保留意见（新增，从W6升级）
- **指标**：审计意见类型
- **判定**：保留意见(Qualified Opinion)
- **处理**：直接否决
- **升级理由**：烟蒂策略100%建立在"资产负债表是真实准确的"假设上。保留意见通常意味着存货查不清、应收账款无法函证、关联方不明，账本基础已被动摇，不允许侥幸

## 扫描流程

1. **第一轮：VETO项扫描**
   - 按V1-V12顺序检查
   - 发现任一VETO项 → 标记"一票否决" → 终止分析

2. **第二轮：WARNING项扫描**
   - 无VETO项时进行
   - 记录所有WARNING项
   - 计算扣分总分(基础分10分)

3. **第三轮：加分项评估**
   - 国企背景：+1分
   - 子公司持股价值明显：+1分
   - 分红记录良好(股息率>5%)：+1分

## 评级标准

| 总分 | 评级 | 说明 |
|------|------|------|
| ≥ 10分 | A | 优质烟蒂股，资产质量高 |
| 8-9分 | B+ | 良好烟蒂股，需关注个别风险 |
| 6-7分 | B | 一般烟蒂股，需充分尽调 |
| 4-5分 | C | 较差烟蒂股，风险较高 |
| < 4分 | D | 不建议投资 |

## 使用示例

**示例：某港股公司 Fact Check 扫描**
```
VETO项扫描：
- V1: 无持续经营不确定性
- V2: 标准无保留意见
- V7: 受限现金占比18% → 未触及否决线
- V12: 不适用(非保留意见)

WARNING项扫描：
- W1: (商誉+无形资产)/净资产 = 22% → -1分
- W5: 关联方往来占比12% → -2分
- W7: 受限现金占比18% → -1分

加分项：
- 国企背景 → +1分

总分：10 - 1 - 2 - 1 + 1 = 7分
评级：B
```
