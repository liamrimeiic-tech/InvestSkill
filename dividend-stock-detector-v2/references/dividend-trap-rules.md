## 概览

本参考文档定义了 12 项红利陷阱扫描规则，分为 VETO（一票否决）和 WARNING（扣分预警）两类。VETO 项一旦触发，直接否决标的；WARNING 项触发后需综合评估，可能降级评级。

## VETO 项（一票否决）

### V1: 母公司分配能力双杀

**定义**：
母公司单体报表显示分配能力存在致命缺陷。

**判断条件（满足任一即触发）**：
1. 母公司单体未分配利润 ≤ 0
2. 拟分配总额 > 母公司单体货币资金

**风险逻辑**：
- 条件 1：违反公司法关于利润分配的前提条件
- 条件 2：账面富贵，资金实际被困在子公司或海外，无法汇出母公司用于分红

**示例**：
```json
{
  "parent_unallocated_profit": -5.2,  // 亿元
  "parent_cash": 8.0,
  "total_distribution": 12.0  // > parent_cash，触发 V1
}
```

---

### V2: 借贷分红/回购实锤

**定义**：
在经营现金流不足以支撑分红的情况下，通过举债维持高分红或高溢价回购，存在利益输送嫌疑。

**判断条件（同时满足）**：
1. TSR 覆盖率 < 0.5
2. 有息负债同比剧增（定义：同比增长 > 30%）
3. 仍维持高分红（股息率 > 行业平均 150%）或高溢价回购（回购价 > 市价 10%）

**风险逻辑**：
- 通过"左手借钱，右手分红"的方式将公司利益输送给大股东
- 典型手法：在股价低迷时高溢价回购，实质是大股东套现

**示例**：
```json
{
  "tsr_coverage": 0.35,  // < 0.5
  "debt_yoy_growth": 45.2,  // %
  "dividend_yield": 8.5,  // %，远高于行业平均 4%
}
```

---

### V3: 利润表严重透支

**定义**：
连续多年分红金额超过盈利能力，靠消耗历史积累维持分红。

**判断条件（同时满足）**：
1. 总分配额 > 归母净利润的 120%
2. 该情况持续 2 年及以上
3. 无重大资产出售或一次性收益补充

**风险逻辑**：
- 超额分红会侵蚀净资产，长期不可持续
- 可能是"温水煮青蛙"式的价值毁灭

**示例**：
| 年份 | 归母净利润 (亿) | 总分配额 (亿) | 分配/利润比 |
|------|----------------|---------------|-------------|
| 2023 | 30 | 42 | 140% |
| 2024 | 28 | 40 | 143% |

---

### V4: 强周期极值陷阱

**定义**：
强周期行业利润处于 10 年高位，但周期拐点已现，分红不可持续。

**判断条件（同时满足）**：
1. 标的属于强周期行业（钢铁、煤炭、有色金属、航运、化工等）
2. 当前归母净利润处于近 10 年最高 20% 分位区间
3. 主导产品现货价格较近期高点回撤 > 20%

**风险逻辑**：
- 周期顶部的高分红是"夕阳红包"，不可持续
- 产品价格下跌将导致利润和分红同步暴跌

**适用行业清单**：
- 钢铁：螺纹钢、热轧卷板
- 煤炭：动力煤、焦煤
- 有色金属：铜、铝、锌
- 航运：BDI 指数
- 化工：MDI、PTA、PVC

---

### V5: 监管红线触底（金融专用）

**定义**：
金融企业资本充足率逼近监管底线，未来必须削减分红补充资本金。

**判断条件（满足任一即触发）**：
1. 银行：核心一级资本充足率 (CET1) < 监管要求 + 1.5%
2. 保险：偿付能力充足率 < 150%
3. 证券：净资本 / 各项风险资本准备之和 < 120%

**风险逻辑**：
- 监管要求是硬约束，违反将面临业务限制
- 补充资本金的优先级高于分红

**示例（银行）**：
```json
{
  "cet1_ratio": 8.2,  // %
  "regulatory_requirement": 7.5,  // %
  "safety_margin": 0.7  // < 1.5%，触发 V5
}
```

## WARNING 项（扣分预警）

### W1: 重大资本开支侵蚀

**定义**：
未来 1-2 年有大规模资本支出计划，将大幅消耗现金流，影响分红可持续性。

**判断条件**：
已披露的资本支出计划 > 当前账面货币资金的 50%

**影响逻辑**：
- 大额 CAPEX 会挤压分红空间
- 需评估项目回报率和回收周期

---

### W2: 质押套现嫌疑

**定义**：
大股东高比例质押股权，且突然发放远超历史的特别股息。

**判断条件（同时满足）**：
1. 大股东质押率 > 70%
2. 当年股息总额 > 过去 3 年平均股息的 200%

**影响逻辑**：
- 高质押率 + 高分红 = 大股东套现缓解平仓压力
- 中小股东可能成为"接盘侠"

---

### W3: 虚假回报（扩表对冲）

**定义**：
一边高额分红，一边大规模增发融资，净效果是"左手发钱，右手要钱"。

**判断条件（满足任一）**：
1. 当年现金分红 > 归母净利润的 50%，且当年定增融资额 > 现金分红额
2. 当年现金分红 > 归母净利润的 50%，且配股融资额 > 现金分红额

**影响逻辑**：
- 分红只是"过账"，中小股东承担融资成本
- 真实回报率为负

---

### W4: 异常畸高收益率

**定义**：
真实股东收益率 > 15%，但基本面恶化，属于被动高息。

**判断条件（同时满足）**：
1. Shareholder Yield > 15%
2. 当期归母净利润同比下降 > 20%
3. 主营业务收入同比下降 > 10%

**影响逻辑**：
- 高收益率源于股价暴跌，非公司主动回馈
- 基本面恶化可能导致分红削减

---

## 判断逻辑示例

### 完整判断流程

```python
def scan_traps(data):
    """
    data 包含所有财务指标和行业信息
    返回 VETO 和 WARNING 列表
    """
    vetos = []
    warnings = []

    # V1: 母公司分配能力双杀
    if data["parent_unallocated_profit"] <= 0:
        vetos.append("V1: 母公司未分配利润 ≤ 0")
    elif data["total_distribution"] > data["parent_cash"]:
        vetos.append("V1: 分配额 > 母公司现金")

    # V2: 借贷分红
    if (data["tsr_coverage"] < 0.5 and
        data["debt_yoy_growth"] > 30 and
        data["dividend_yield"] > data["industry_avg_yield"] * 1.5):
        vetos.append("V2: 借贷分红实锤")

    # V3: 利润表透支
    if (all(year["dist/profit_ratio"] > 1.2 for year in data["last_3_years"]) and
        not data["has_oneoff_gain"]):
        vetos.append("V3: 利润表严重透支")

    # V4: 周期陷阱
    if (is_cyclical(data["industry"]) and
        is_at_decile(data["profit"], data["10y_profit_history"], 0.8) and
        data["product_price_decline"] > 0.2):
        vetos.append("V4: 强周期极值陷阱")

    # W4: 畸高收益率
    if (data["shareholder_yield"] > 15 and
        data["profit_yoy_decline"] > 0.2 and
        data["revenue_yoy_decline"] > 0.1):
        warnings.append("W4: 异常畸高收益率")

    return {"VETO": vetos, "WARNING": warnings}
```

## 约束与注意事项

1. **数据来源优先级**：
   - 优先使用年报官方数据
   - 财报注释中的回购明细是判断"注销式回购"的唯一依据
   - 母公司单体报表必须从公司公告中获取，不可用合并报表推算

2. **行业特殊性**：
   - 金融业不适用 FCF 模型，V2 和 V4 需基于监管指标判断
   - 地产行业需扣除预售资金监管受限部分
   - REITs 的分红逻辑不同，不适用本规则

3. **时间窗口**：
   - V3 要求连续 2 年透支，单年透支仅触发 WARNING
   - V4 的价格回撤需观察 3-6 个月趋势，避免短期波动误判
