## 概览

本文档详细说明总股东回报 (TSR) 的计算方法，包括核心概念、通用公式、行业特定调整和边界条件处理。

## 核心概念定义

### 1. 总股东回报 (Total Shareholder Return, TSR)

**定义**：股东通过现金分红和股份回购两种方式获得的总回报率。

**数学表达**：
```
TSR = (现金分红 + 注销式回购) / 总市值
```

**关键区分**：
- **注销式回购**：回购后注销股份，减少总股本，提升 EPS，计入 TSR
- **员工持股计划回购**：回购后用于员工激励，本质是费用转移，不计入 TSR

---

### 2. TSR 覆盖率

**定义**：真实自由现金流对总分配额的覆盖倍数，衡量分红可持续性。

**数学表达**：
```
TSR 覆盖率 = 真实自由现金流 / 总分配额
```

**健康标准**：
- > 1.5：非常健康，分红可持续性强
- 1.2 - 1.5：健康
- 0.8 - 1.2：需关注，可能压缩分红
- < 0.8：高风险，可能削减分红
- < 0.5：危险，存在借贷分红嫌疑

---

### 3. 真实股东收益率 (Shareholder Yield)

**定义**：以股价为分母的总回报率，与传统股息率的区别在于包含回购。

**数学表达**：
```
Shareholder Yield = 总分配额 / 总市值 × 100%
传统股息率 = 现金分红 / 总市值 × 100%
```

**优势**：
- 更全面反映股东回馈
- 对"以回购代分红"的公司更公平
- 跨公司比较更有意义

---

### 4. 母公司现金流转摩擦率

**定义**：总分配额占母公司单体货币资金的比例，衡量资金跨境/跨实体流动的摩擦成本。

**数学表达**：
```
母公司现金流转摩擦率 = 总分配额 / 母公司单体货币资金
```

**判断标准**：
- < 0.5：资金流动顺畅，无障碍
- 0.5 - 1.0：存在一定摩擦，需关注
- > 1.0：致命警告，资金可能被困在子公司或海外

---

## 通用版计算公式

### 基础模型（适用于一般制造业、消费业等）

**真实自由现金流 (Real FCF)**：
```
Real FCF = 经营活动现金流净额 (CFO) - 资本支出 (CAPEX)
```

**TSR 核心指标**：
```
总分配额 = 年度现金分红 + 年度注销式回购金额
TSR 覆盖率 = Real FCF / 总分配额
Shareholder Yield = 总分配额 / 总市值 × 100%
```

**计算步骤**：
1. 从现金流量表提取 CFO（建议使用近 3 年平均值）
2. 从现金流量表提取 CAPEX（建议使用近 3 年平均值）
3. 从利润分配方案提取现金分红总额
4. 从股份回购公告提取注销式回购金额（严格区分用途）
5. 计算总分配额
6. 计算各项 TSR 指标

## 行业特定调整

### 房地产行业

**特殊性**：
- 预售资金受监管，开发商不能自由支配
- 拿地支出是刚性资本开支，不可压缩

**调整公式**：
```
Real FCF (地产版) = CFO - 预售资金监管受限部分 - 刚性拿地支出
```

**参数获取**：
- 预售资金监管受限部分：从年报"资金使用受限情况"注释中提取
- 刚性拿地支出：从经营计划或土地储备分析中提取

---

### 金融行业（银行、保险、证券）

**特殊性**：
- 自由现金流概念不适用
- 监管资本充足率是硬约束

**替代指标**：
- **银行**：核心一级资本充足率 (CET1)、拨备覆盖率
- **保险**：偿付能力充足率、内含价值增长
- **证券**：净资本、风险控制指标

**判断逻辑**：
```
银行分红能力判断：
- CET1 > 监管要求 + 2.5%：分红能力极强
- CET1 > 监管要求 + 1.5%：分红能力健康
- CET1 ≤ 监管要求 + 1.5%：触发 V5
```

## 边界条件处理

### 1. 零值与负值处理

**情况 A：CFO < 0**
- 计算 TSR 覆盖率时，Real FCF 取绝对值
- 触发 WARNING：连续 2 年 CFO < 0

**情况 B：母公司货币资金 = 0**
- 母公司现金流转摩擦率设为 ∞
- 自动触发 V1（母公司分配能力双杀）

**情况 C：市值 = 0（不可能，但需防错）**
- Shareholder Yield 设为 None
- 抛出数据异常警告

---

### 2. 缺失数据降级原则

**数据缺失降级链**：
```
完整数据（5 年） → 3 年数据 → 单年数据 → 无法计算
```

**降级处理**：
- 无母公司报表：强制生成 WARNING，评级降一级
- 无回购明细：注销式回购金额设为 0，仅计算现金分红
- 无 CAPEX 明细：使用折旧摊销额替代（保守估计）

---

### 3. 极值过滤

**异常值检测**：
- TSR 覆盖率 > 10：设为 10（避免单年异常影响）
- Shareholder Yield > 30%：设为 30%，触发 W4 预警
- 母公司现金流转摩擦率 > 10：设为 10，触发 V1

---

## 计算示例

### 示例 1：一般制造业（健康案例）

**输入数据**：
```json
{
  "industry_type": "general",
  "cfo_avg": 50,
  "capex_avg": 20,
  "net_profit": 40,
  "total_dividend": 15,
  "cancellation_buyback": 5,
  "parent_unallocated_profit": 100,
  "parent_cash": 18,
  "market_cap": 300
}
```

**计算过程**：
```
Real FCF = 50 - 20 = 30 亿元
总分配额 = 15 + 5 = 20 亿元
TSR 覆盖率 = 30 / 20 = 1.5 (健康)
Shareholder Yield = 20 / 300 × 100% = 6.67% (合理)
母公司现金流转摩擦率 = 20 / 18 = 1.11 (> 1，触发 WARNING)
```

**输出结果**：
```json
{
  "metrics": {
    "total_distribution": 20,
    "shareholder_yield": 6.67,
    "real_fcf": 30,
    "tsr_coverage": 1.5,
    "parent_cash_friction": 1.11
  },
  "warnings": [
    {
      "type": "WARNING",
      "code": "V1",
      "message": "母公司现金流转摩擦率 1.11 > 1，资金可能被困在子公司"
    }
  ]
}
```

### 示例 2：强周期行业（陷阱案例）

**输入数据**：
```json
{
  "industry_type": "general",
  "sector": "steel",
  "cfo_avg": 80,
  "capex_avg": 30,
  "net_profit": 60,
  "total_dividend": 36,
  "cancellation_buyback": 0,
  "product_price_decline": 0.25,
  "profit_10y_percentile": 0.95
}
```

**计算过程**：
```
Real FCF = 80 - 30 = 50 亿元
总分配额 = 36 + 0 = 36 亿元
TSR 覆盖率 = 50 / 36 = 1.39 (表面健康)
Shareholder Yield = 36 / (60 × 15) × 100% = 4% (假设 PB=1.5)
利润分位数 = 0.95 (近 10 年最高 5%)
价格回撤 = 25%
```

**智能体判断**：
```
触发 V4: 强周期极值陷阱
- 钢铁行业处于 10 年高位
- 螺纹钢价格回撤 25%
- 虽然 TSR 覆盖率健康，但周期拐点已现
```

---

### 示例 3：金融行业（监管约束）

**输入数据**：
```json
{
  "industry_type": "finance",
  "sector": "bank",
  "cet1_ratio": 8.2,
  "regulatory_cet1_requirement": 7.5,
  "provision_coverage": 180,
  "net_profit": 50,
  "total_dividend": 25
}
```

**计算逻辑**：
```
不使用 FCF 模型
判断依据：CET1 安全边际
安全边际 = 8.2% - 7.5% = 0.7%
健康标准：安全边际 > 1.5%
当前安全边际 = 0.7% < 1.5%
```

**输出结果**：
```json
{
  "industry_type": "finance",
  "metrics": {
    "cet1_safety_margin": 0.7,
    "dividend_cap_ratio": 50
  },
  "warnings": [
    {
      "type": "VETO",
      "code": "V5",
      "message": "CET1 安全边际 0.7% < 1.5%，逼近监管红线，未来必须削减分红"
    }
  ]
}
```

## 约束与注意事项

1. **数据质量要求**：
   - 必须使用审计后的年报数据
   - 回购明细需从股东大会决议或公告中提取
   - 母公司报表不可用合并报表推算

2. **时间维度**：
   - CFO 和 CAPEX 建议使用 3 年移动平均
   - 单年数据可能受季节性或一次性事件影响
   - TSR 覆盖率趋势比单年数值更重要

3. **行业适配**：
   - 通用模型仅适用于非金融、非地产企业
   - 金融企业必须切换至监管指标模型
   - REITs、MLPs 等特殊结构需单独评估

4. **防幻觉机制**：
   - 未明确回购用途的，一律不计入注销式回购
   - 母公司报表缺失的，必须强制生成 WARNING
   - 极值数据需二次校验来源
